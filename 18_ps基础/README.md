## 菜单栏介绍

* ctrl+R 启用标尺

* 文件菜单
* 新建 	Ctrl+N 
* 打开	Ctrl+O
* 打开文档两种方式：
* A 文件菜单选项  打开文档 （快捷键Ctrl+O ）
* B 直接拖拽文档至ps中  
* 保存 Ctrl+S 
* 图片格式 JPEG BMP  PNG GIF
* 原文件格式 PSD
* 另存为 Ctrl+Shift+S 可以存为其他的格式
* 关闭 Ctrl+W 
* 编辑
* 图层
* 文字
* 选择
* 滤镜
* 3D
* 试图
* 窗口
* 帮助
